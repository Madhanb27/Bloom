# Bloom 

A premium botanical lifestyle website focused on luxury home décor and indoor plants.

## Live Demo

🔗 https://bloom-living.vercel.app/

## Overview

Bloom is a modern botanical lifestyle website designed to create a calm, elegant, and premium digital experience. The project combines luxury branding, responsive design, immersive visuals, and smooth interactions to showcase plants and home décor products.

## Features

* Responsive desktop and mobile design
* Luxury botanical brand identity
* Product showcase section
* Lifestyle inspiration gallery
* Customer testimonials
* Newsletter subscription section
* Smooth interactions and animations
* GitHub + Vercel deployment workflow

## Tech Stack

* Figma
* HTML5
* CSS3
* JavaScript
* GSAP
* Git
* GitHub
* Vercel

## Design Goals

* Create a premium botanical brand experience
* Maintain visual consistency across all sections
* Deliver a clean and modern responsive layout
* Focus on typography, imagery, and whitespace

## Deployment

The project is deployed on Vercel and automatically updates whenever changes are pushed to GitHub.
